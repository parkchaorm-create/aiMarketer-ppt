말로 그리는 캔버스

  • 01⏱️ 구성 · 한 호흡으로 완주→
    HOOK → CONCEPT → 실습 → RECAP 순으로 구성됩니다. 실습 비중이 크니 노션 튜토리얼을 미리 열어두십시오.
  • 02🎨 비유 · 말로 그리는 디자인 캔버스→
    문장 하나만 던지면 AI가 대신 그려줍니다. 붓 대신 말로 페이지를 그린다고 생각하시면 됩니다.
  • 03🎯 목표 · 시안 1개 화면에 띄우기→
    완벽한 결과물보다 '내가 말한 대로 만들어졌구나' 확인이 오늘의 목표입니다. 다운로드·배포는 다음 회차에서 다룹니다.
  • 04🧰 도구 · Stitch (Google Labs, 무료)→
    구글이 무료로 공개한 AI 웹 UI 디자인 도구입니다. 계정만 있으면 바로 쓸 수 있습니다.

코딩 한 줄 없이, 웹페이지가 나올까요?

  • 01💬 문장 하나로 · 진짜 웹 시안이 뽑힐까요?→
    Stitch(스티치)는 문장 하나만 입력하면 웹 페이지 시안(실제 배포 전 단계의 화면 밑그림)을 자동으로 만들어 주는 구글의 AI 디자인 도구입니다.
  • 02👀 오늘은 보는 단계 · 다운로드는 아직일까요?→
    시안을 생성해서 보는 단계까지만 진행합니다. 다운로드·공개 배포(Deploy: 만든 결과물을 인터넷에 올려 공개하는 것)는 다음 회차에서 다룰 예정입니다.
  • 03🎯 말한 대로 · 진짜 그렇게 나왔을까요?→
    완벽을 기대하지 않으셔도 됩니다. 시안 1개가 화면에 떠서 '내가 말한 대로 됐구나' 확인하시는 게 오늘의 목표입니다.

Stitch · 자연어 UI 도구

  • 01🌐 stitch.withgoogle.com · 자연어 UI 생성→
    Stitch는 자연어(일상적으로 쓰는 말)로 웹 UI(User Interface: 사용자가 보고 조작하는 화면) 디자인을 만드는 도구입니다. 주소는 stitch.withgoogle.com이고 브라우저에서 바로 접속 가능합니다.
  • 02🆓 Google Labs 무료 · 하루 제한은 충분→
    Google Labs(구글 랩스: 구글이 시범 서비스를 공개하는 실험 공간)에서 무료로 제공됩니다. 1일 생성 횟수 제한이 있지만 오늘 실습에는 충분하니 걱정 마십시오.
  • 03🇰🇷 한국어 OK · 영어가 조금 더 일관적→
    한국어 입력도 가능합니다. 다만 영어로 입력하면 조금 더 일관된 결과가 나오는 경향이 있습니다.

💬 그대로 읽으세요
Stitch 주소는 stitch.withgoogle.com입니다. 자연어로 웹 UI 디자인을 만드는 도구예요. Google Labs에서 무료 베타로 제공됩니다. 2026년 4월 기준이에요. 1일 생성 횟수 제한이 있지만 오늘 실습에 충분합니다. 한국어 입력도 가능하고, 영어로 입력하면 조금 더 일관된 결과를 내는 경향이 있어요. 편하신 언어로 시작하시면 됩니다.

  • 01👨‍🏫 강사 시연 · 이어서 개별 실습→

결과 공유 · 슬랙 업로드

  • 01📤 시안 캡처 · #수업실습 채널 업로드→
    가능하신 분은 시안 캡처를 슬랙 #수업실습 채널에 업로드해 주시면 좋습니다. 강제는 아닙니다.
  • 02👀 다른 사람 시안 · 구경도 좋은 학습→
    다른 수강생의 시안을 구경하시는 것도 좋은 학습입니다. 같은 프롬프트라도 결과가 다 달라서 흥미롭습니다.
  • 03✅ 오늘까지 · 시안 확인, 내보내기는 다음→
    오늘은 시안 확인까지만 진행합니다. HTML(웹 페이지를 구성하는 기본 언어) 내보내기나 공개 URL(인터넷 주소) 생성은 다음 회차 또는 심화 과제에서 다룹니다.
