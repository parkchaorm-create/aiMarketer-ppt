Phase 3 · 자동화 기초 (10~15회)

  • 01⚙️ 10~15회 · 자동화의 세계 진입→
    10회부터는 자동화(사람이 매번 손대지 않아도 도구가 스스로 일하게 만드는 것)의 세계로 들어갑니다. 수작업을 줄이고 도구가 알아서 돌게 만드는 단계예요.
  • 02💻 Claude Code · 터미널 기반 도구→
    터미널(Terminal: 컴퓨터에 명령어를 글로 입력해 조작하는 검은 창) 기반 도구인 Claude Code(클로드 코드: 터미널에서 명령어로 자동화를 설계하는 앤트로픽의 에이전트)를 활용해요. CLAUDE.md(클로드닷엠디: Claude Code의 동작 규칙을 적어 두는 설정 파일) 설정과 자동 리퍼포징(Repurposing: 한 콘텐츠를 여러 채널용으로 변형하는 것)까지 다룹니다.
