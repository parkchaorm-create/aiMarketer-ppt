2022년 그날, 세상이 얼마나 바뀐 걸까요?

  • 01📅 ChatGPT 등장 전후 · 일상이 달라졌을까요?→
    생성형 AI(Generative AI: 글·이미지·영상 같은 새 결과물을 직접 만들어 주는 AI)는 2022년 11월 ChatGPT(챗지피티: 오픈AI의 대화형 AI) 공개를 기점으로 대중에게 알려졌습니다. 그 전과 후, 일상이 달라졌는지 함께 짚어 봅니다.
  • 02🌊 3년에 4단계 · 따라잡을 수 있을까요?→
    이후 약 3년 동안 대화형(사람과 글로 대화하듯 주고받는 방식)·멀티모달(Multimodal: 텍스트·이미지·음성·영상을 함께 다루는 방식)·에이전트(Agent: 시킨 작업을 스스로 처리하는 AI)·Co-work(Co-work: 사람과 AI가 같은 작업 공간에서 함께 작업하는 방식)까지 네 단계의 큰 변화가 있었습니다. 빨라 보이지만 단계만 잡으면 이해하실 수 있습니다.
  • 03🧑‍💼 마케터 업무 · 나는 뭐가 달라질까요?→
    기술이 단계를 밟는 동안 마케터의 업무 방식도 같이 바뀌었습니다. 오늘 그 흐름을 정리합니다.

대화형 · 2022.11~2023

  • 01🚀 2022.11.30 · ChatGPT(GPT-3.5) 공개→
    2022년 11월 30일 OpenAI(오픈AI: ChatGPT를 만든 회사)가 ChatGPT(당시 탑재 모델은 GPT-3.5)를 공개했습니다. 생성형 AI 대중화의 출발점이 된 사건입니다.
  • 02📈 5일 만에 100만 · 3개월 만에 1억→
    출시 5일 만에 사용자 100만 명(OpenAI 공식 발표), 2023년 2월에는 월간 활성 사용자(MAU: Monthly Active Users, 한 달에 한 번이라도 서비스를 쓴 사용자 수) 1억 명을 돌파했습니다. 당시 소비자 앱 최단기 1억 달성으로 보도됐습니다.
  • 03🤖 Claude · Bard 합류 · 대화형 3종 체제→
    같은 해 Anthropic(앤트로픽: Claude를 만든 회사)의 Claude(클로드: 앤트로픽의 대화형 AI), 구글의 Bard(바드: 구글의 초기 대화형 AI, 이후 Gemini로 개편)가 공개되며 주요 대화형 AI 3종 체제가 형성됐습니다.
  • 04📝 마케터 관점 · 텍스트 초안 보조→
    마케터는 블로그 초안 같은 텍스트 초안 작성에 AI를 보조로 쓰기 시작한 단계입니다.

멀티모달 · 2024년

  • 01🎨 GPT-4o · Claude 3.5 · Gemini 1.5→
    한 모델이 텍스트·이미지·음성·영상을 함께 처리하는 멀티모달 모델이 2024년에 상용화됐습니다.
  • 02🖼️ Midjourney v6 · Runway Gen-3 · Suno→
    Midjourney(미드저니: 대표적인 AI 이미지 생성 서비스) v6, Runway(런웨이: AI 영상 생성 서비스) Gen-3, Suno(수노: AI 음악 생성 서비스) 등 이미지·영상·음악 생성 서비스가 대중화된 시기입니다.
  • 03📸 마케터 관점 · 시각 콘텐츠 본격 활용→

에이전트 · 2024.10~2025

  • 01🖱️ 2024.10 · Computer Use 공개→
    2024년 10월 Anthropic이 Computer Use(컴퓨터 유즈: AI가 사람 대신 화면을 보고 마우스·키보드를 조작하는 기능)를 공개했습니다. Claude가 화면을 보고 마우스·키보드를 직접 제어하는 기술입니다.
  • 02🔌 2024.11 · MCP 개방형 표준 발표→
    2024년 11월 Anthropic이 MCP(Model Context Protocol: AI가 외부 서비스와 연결되도록 정해 놓은 개방형 표준)를 발표했습니다. 노션·깃허브·CRM(Customer Relationship Management: 고객 관계 관리 도구) 같은 외부 서비스와 AI를 연동하는 개방형 표준입니다.
  • 03🌐 Mariner · Operator · 웹 자동화 합류→
    2024년 12월 구글 DeepMind(딥마인드: 구글의 AI 연구 조직) Project Mariner(마리너: 구글의 웹 자동 탐색 에이전트), 2025년 1월 OpenAI Operator(오퍼레이터: 오픈AI의 웹 과업 자동 수행 에이전트)가 공개되며 웹 과업 자율 수행 에이전트가 늘었습니다.
  • 04🤝 마케터 관점 · 시키면 알아서 수행→
    "시키면 일부 작업을 알아서 수행하는" 에이전트형(Agent형: 사람의 지시를 받아 스스로 일을 처리하는 형태) AI를 마케터도 쓰기 시작한 단계입니다.

💬 그대로 읽으세요
3단계는 자동화 에이전트의 등장입니다. 대화형 AI가 외부 도구를 직접 조작하기 시작한 시기예요. 2024년 10월 Anthropic Computer Use가 공개됐습니다. Claude가 화면을 보고 마우스·키보드를 직접 움직이는 기능이에요. 바로 다음 달 11월에 Anthropic이 MCP, Model Context Protocol을 발표했습니다. AI가 노션·깃허브·CRM 같은 외부 서비스와 연동되는 개방형 표준이에요. 본 과정 4 Phase, 그러니까 16~25회에서 MCP 기반 GHL 연동을 실습합니다. 12월에는 구글 DeepMind의 Project Mariner, 2025년 1월에는 OpenAI Operator가 공개됐어요. 웹 과업을 자율적으로 수행하는 에이전트들이 줄줄이 합류한 겁니다. 마케터 관점에서는 시키면 일부 작업을 알아서 수행하는 에이전트형 AI를 활용하기 시작한 단계입니다.

Co-work · 2025~2026

  • 01🖥️ Claude Projects · Artifacts 협업→
    대화창 옆에 작업 창이 열려 코드·문서·디자인을 AI와 함께 편집합니다. 사람과 AI가 같은 캔버스(Canvas: 공동 작업이 펼쳐지는 빈 화면·영역)에서 작업하는 방식이 보편화됐습니다.
  • 02📝 ChatGPT Canvas · Projects 유사 기능→
    OpenAI도 Canvas와 Projects로 비슷한 공동 편집 환경을 제공합니다.
  • 03🧩 구글 · Flow · Stitch · AI Studio · Gemma 4→
    2025년 5월 Google I/O(구글 아이오: 구글의 연례 개발자 콘퍼런스)에서 발표된 통합 도구 세트입니다. Flow(플로우: 구글의 AI 영상 제작 도구), Stitch(스티치: 구글의 AI UI 디자인 도구), AI Studio(AI 스튜디오: 구글의 AI 개발 스튜디오), 2026년 4월 공개된 오픈소스(Open Source: 설계·코드를 무료로 공개해 누구나 쓸 수 있게 한 것) Gemma 4(젬마 4: 구글의 오픈소스 AI 모델)까지 포함됩니다.
  • 04⌨️ Claude Code · 터미널 자동화 에이전트→
    Anthropic이 2024년 10월 공개한 Claude Code(클로드 코드: 터미널에서 명령어로 자동화를 설계하는 앤트로픽의 에이전트)는 터미널(Terminal: 컴퓨터에 명령어를 글로 입력해 조작하는 검은 창) 기반 자동화 에이전트입니다. n8n(엔에이트엔: 노코드 자동화 도구)·Make(메이크: 또 다른 노코드 자동화 도구)로 만들던 워크플로우(Workflow: 여러 단계의 작업 흐름)를 자연어(일상적으로 쓰는 말) 지시로 설계할 수 있어 최근 활용이 늘고 있다는 평가가 있습니다.
  • 05🤝 마케터 관점 · 보조에서 협업 도구로→
    AI가 "보조 도구"에서 "협업 도구"로 역할이 확대되는 단계입니다.

📝 강사 노트
💬 그대로 읽으세요
4단계는 공동작업, Co-work 시대입니다. 2025년부터 2026년 현재까지의 모습이죠. 사람과 AI가 같은 캔버스에서 함께 작업하는 방식이 보편화됐습니다. Claude Projects·Artifacts는 대화창 옆에 작업 창이 열려서 코드·문서·디자인을 AI와 함께 편집할 수 있어요. ChatGPT도 Canvas와 Projects로 유사 기능을 제공합니다. 구글은 2025년 5월 Google I/O에서 통합 도구를 발표했어요. Flow는 영상 제작, Stitch는 자연어 UI 디자인, AI Studio는 개발 환경. 오늘 실습에서 Stitch를 일부 사용할 겁니다. Claude Code도 이 흐름에서 중요한데, 2025년에 앤트로픽이 공개한 터미널 기반 자동화 에이전트예요. n8n·Make 같은 노코드 도구로 만들던 워크플로우를 자연어 지시로 설계할 수 있어서 최근 활용이 늘고 있다는 평가가 있습니다. 본 과정 3~4 Phase에서 실습합니다. 마케터 관점에서는 AI가 보조 도구에서 협업 도구로 역할이 확대되는 단계예요.
