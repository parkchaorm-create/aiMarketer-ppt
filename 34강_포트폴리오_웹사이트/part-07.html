#수업실습 · 세 파일 · 태그 붙여서

  • 01 🌐 파일 1 · 라이브 URL .txt →
    팀의 공식 URL 1줄이 담긴 텍스트 파일. 파일명 예: [34강][A] 푸르트_라이브URL.txt.
  • 02 📸 파일 2 · 스크린샷 3장 →
    홈·Works·Contact 각 1장씩 라이브 사이트 스크린샷. 파일명 예: [34강][C] 손결_스크린샷_홈.png.
  • 03 📝 파일 3 · RACI 회고 카드 →
    RACI 회고 카드 + 파이널 누적 자산 표 한 장. 팀당 1장. 예: [34강][B] 슬로우템포_RACI회고.md.
  • 04 🏷️ 태그 · [34강][클라이언트] →
    파일명 앞에 [34강][A/B/C/D]를 붙이시면 조교가 바로 확인 가능. 1~33강과 같은 규칙입니다.
