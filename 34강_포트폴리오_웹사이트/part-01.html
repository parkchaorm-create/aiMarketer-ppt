Designer · Integrator · Content

  • 01 🎨 Designer · R · Stitch 시안 →
    R(Responsible · 실행 책임) — Stitch에서 4~5페이지 시안 생성 · 텍스트·컬러 수정. 쓰는 도구는 Stitch 단 하나.
  • 02 ⚙️ Integrator · A · Claude Code →
    A(Accountable · 승인 책임) — Claude Code에서 HTML 수정 · GHL 임베드 삽입 · Git 커밋·푸시 · Web Deploy(웹 배포).
  • 03 📦 Content · C · 재료 공급 →
    C(Consulted · 자문 책임) — 4·9·32강 재료 공급 · 최종 검수. 구글 드라이브와 팀 공유 문서가 주 활동 무대입니다.
  • 04 👥 3·4인 팀 · 배분 원칙 →
    3인이면 1인 1역할 · 4인이면 Designer 2인으로 분할. I(Informed · 공유 대상)는 팀 전원 · 슬랙 #수업실습으로 진행 공유.
