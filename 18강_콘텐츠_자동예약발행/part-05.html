02 / 07

Post Now · 혹시 누르셨나요?

  • 01 ⚠️ 실수 1순위 · Post Now 즉시 발행 →
    예약이 처음인 분이 가장 자주 누르는 버튼이 Post Now(즉시 발행)예요. 반드시 Schedule for Later(미래 예약)를 선택하세요.
  • 02 🧭 예약 시각 · 수업 종료 30분~1시간 뒤 →
    오늘 실습 예약 시각은 수업 종료 시점 기준 30분~1시간 뒤로 잡겠습니다. 실제 발행 여부는 수업 뒤 편한 시점에 확인.
  • 03 🔧 실수 복구 · 당황 금지 · 방법 있음 →
    실수로 즉시 발행되어도 Calendar에서 해당 게시물 Delete · Blog는 Status를 Draft로 전환해 복구 가능. 마지막에 안내드립니다.

03 / 07

Social Planner · 6단계 골격

  • 01 🚪 진입 · Marketing → Social Planner →
    좌측 메뉴 Marketing 안에 Social Planner. New Post(또는 Create New Post) 버튼으로 새 예약 창 진입.
  • 02 ✅ 채널 · IG + FB 함께 체크 →
    채널 선택 영역에서 인스타그램과 페이스북 두 개에 동시 체크. "Customize for Each Post" 토글이 켜진 상태인지 확인.
  • 03 📝 본문 · 채널별 개별 입력 →
    인스타 칸에 캡션·해시태그·이미지 · 페이스북 칸에 텍스트·링크. Part 04에서 만든 변형안을 채널별로 개별 붙여넣기.
  • 04 ⏰ Schedule for Later · 시각 지정 →
    수업 종료 30분~1시간 뒤로 설정. 우측 상단 스케줄 버튼 → 예약 확정. Calendar 탭으로 돌아가 블록 등장 확인.

04 / 07

Blog · 6단계 · 메타까지

  • 01 🚪 진입 · Sites → Blogs → New Post →
    좌측 Sites 메뉴 안 Blogs. 기존 블로그 선택 또는 New Blog 생성 후 New Post로 진입.
  • 02 📝 본문 · 800~1500자 붙여넣기 →
    Part 04에서 생성한 블로그 제목·부제·본문을 붙여넣기. Save and Continue로 다음 단계(메타 입력)로 이동.
  • 03 🏷️ 메타 · slug·카테고리·SEO →
    URL slug · 카테고리 · 저자 · 키워드 · SEO description · 대표 이미지 alt text. 6개 필드 모두 채우면 검색 유입이 조금 나아집니다.
  • 04 📅 Status = Schedule · 시각 확정 →
    Status를 Schedule로 선택 후 날짜·시간 지정 → Save/Publish. 성공 신호: 목록에서 상태가 "Scheduled"로 표시.

05 / 07

스냅샷 2장 · 업로드까지

  • 01 📷 방법 A · 캡처 2장 →
    Social Planner Calendar 화면 1장 + Blog 게시물 목록의 Scheduled 상태가 보이는 화면 1장. 가장 표준 방법입니다.
  • 02 🖼️ 방법 B · 두 탭 · 통합 1장 →
    브라우저에서 두 탭을 나란히 배치해 한 장으로 캡처해도 무방. 두 모듈이 동시에 보이면 인정.
  • 03 📤 업로드 · 슬랙 #수업실습 →
    업로드 포맷: [18강][A/B/C/D] 3채널예약1건·달력스냅샷. 예: [18강][A] 푸르트 5월 꽃 구독 3채널 예약 · 달력 스냅샷 2장.

실습 B 핵심 · 다섯 줄

  • 01 🧮 3채널 동시 · 액션 2번 구성 →
    Social Planner 예약 1회(IG+FB) + Blog 예약 1회 = 채널 3개 커버. 이 숫자 관계가 실습의 나침반.
  • 02 🎛️ Planner · Customize for Each Post →
    Social Planner는 Customize for Each Post 토글로 채널별 텍스트 다르게 입력. 복붙 금지의 실제 구현 지점.
  • 03 📌 Blog · Schedule 상태 확정 필수 →
    Blog는 Save만 하면 Draft에 머무름. 반드시 Status를 Schedule로 바꾸고 날짜·시간을 지정해야 예약 완료.
  • 04 📸 스냅샷 1~2장 · 업로드 · 포맷 고정 →
    Social Planner Calendar 캡처 + Blog Scheduled 캡처. 슬랙 #수업실습에 [18강][클라이언트] 태그로 업로드.
