세 시연 · 한 흐름

  • 01 💪 시연 ① · Soul ID 결과 확인 →

  • 02 🖼️ 시연 ② · 이미지 참조 I2V →

  • 03 🔁 시연 ③ · 프롬프트 반복 T2V →
    캐릭터 상세 묘사 블록을 3컷에 그대로 복붙해 실시간 생성. 오늘 수강생 실습 방식의 원형을 눈에 새깁니다.

Soul ID · 이미지 참조 · 흐름

  • 01 🔑 Soul ID · Characters 메뉴 →
    힉스필드 상단 Characters 메뉴에 사전 학습된 캐릭터 리스트가 있고, 선택하면 Soul 2.0 프리셋 조합으로 새 장면을 만들 수 있습니다.
  • 02 🎭 Soul · 얼굴 고정 · 배경 변화 →
    같은 캐릭터에 조명·배경·의상만 바꿔 넣으면 3장이 나란히 비교됩니다. 얼굴은 같고 분위기만 바뀌는 감각을 확인합니다.
  • 03 📥 참조 · Create → I2V 업로드 →
    Create 메뉴 → Image to Video에 인물 사진 1장 업로드. 같은 이미지에 카메라 프롬프트만 바꿔 3번 돌립니다.
  • 04 ⚖️ 참조 · 100% 일치는 아님 →
    각도가 바뀔 때 얼굴이 약간 흔들리는 현실적 한계를 함께 확인합니다. "같은 느낌"은 유지되지만 완벽하진 않습니다.

04 / 07

캐릭터 블록 · 3번 복붙

  • 01 🧱 캐릭터 블록 · 4~5줄 영문 →
    나이대·헤어·의상·표정·얼굴 특징을 4~5줄 영문으로 묶은 한 덩어리. 이 블록이 3컷 모두에 그대로 들어갑니다.
  • 02 🎬 컷 1 · Wide · 장면 설정 →
    공간·아침 빛·팔레트를 먼저 깔고 인물이 등장합니다. 카메라는 Static 또는 아주 느린 Pan 정도로 가볍게.
  • 03 🎬 컷 2 · Close-up · 행동 →
    손·표정 근접 디테일. 캐릭터 블록 그대로 + "hands gently adjust" 같은 미세 동작 한 줄을 얹습니다.
  • 04 🎬 컷 3 · Medium · 완성·여운 →
    완성된 모습과 부드러운 미소. Slow pull-back으로 카메라가 뒤로 빠지며 브랜드 감성의 여운을 남깁니다.

Part 03 · 세 줄 정리

  • 01 👀 세 전략 · 눈으로 확인 →
    Soul ID 프리셋 조합 · 이미지 참조 I2V 3회 · 프롬프트 반복 T2V 3회. 각 전략의 실제 화면과 결과 차이를 감각으로 익혔습니다.
  • 02 ⚖️ 완벽보다 · "같은 느낌" →
    프롬프트 반복은 100% 정답이 아닌 80% 정답 도구. 3~5컷 정도까지 쓸 만한 일관성이 나옵니다. 그 이상은 Soul ID 권장.
  • 03 🎯 실습 원형 · 시연 ③번 →
    캐릭터 블록을 3컷 프롬프트의 주인공 자리에 그대로 복붙. Part 05에서 손으로 해 볼 구조가 바로 이겁니다.
