02 / 06

도구 4택1 · 장점 재확인

  • 01🔒 도구 변경 금지 · 중간 교체는 재작업→
    팀 내에서 도구를 바꾸지 말아 주세요. 중간에 바꾸면 재작업 비용이 크거든요.
  • 02🇰🇷 미리캔버스 · 한국어 UI + PDF 원클릭→
    한국어 UI(User Interface · 사용자 조작 화면)라 매뉴얼 템플릿(Template · 미리 만들어 둔 서식)을 검색하기 쉽고, PDF(Portable Document Format · 표준 문서 파일) 저장이 원클릭으로 되는 게 장점이에요.
  • 03🎨 Canva · 영문 템플릿 풍부 + Brand Kit→
    영문 템플릿이 풍부하고 'Brand Kit(브랜드 키트 · 로고·컬러·폰트를 묶어 저장하는 기능)' 개념과 잘 어울려요. 구글 슬라이드는 팀 동시 편집이 가장 매끄럽습니다.
  • 04🤖 Stitch · AI 자동 레이아웃 + 디테일 제한→
    AI가 자동으로 레이아웃(Layout · 요소들의 배치 구조)을 잡아줘서 초안이 빠르지만, 세부 수정은 제한적이에요.

Part 04에서 팀이 고르신 도구를 다시 확인할게요. 팀 내에서 도구를 바꾸지 말아 주세요. 중간에 바꾸면 재작업 비용이 큽니다.

도구별 이 단계에서의 장점을 짚어볼게요.

미리캔버스. 한국어 UI에 매뉴얼 템플릿 검색 용이. PDF 저장 원클릭.

Canva. 영문 템플릿 풍부. Brand Kit 개념과 어울림.

구글 슬라이드. 팀 동시 편집 가장 매끄러움. 파일 다운로드로 PDF.

Stitch. AI 자동 레이아웃으로 빠른 초안. 단, 디테일 수정은 제한적.

추천은 공공기관 수강생에게 가장 친숙한 한국어 UI는 미리캔버스 또는 구글 슬라이드입니다. 팀 내 의견이 갈리면 이 두 개 중에서 고르세요.

라이선스 · 출처 재점검

  • 01🔍 이름 · KIPRIS 3건 재검색 필수→
    브랜드 이름은 KIPRIS(키프리스 · 한국 특허청의 상표·특허 검색 서비스)에서 3건 재검색해요. 참고용 검색이며 법률 자문을 대체하지는 않습니다.
  • 02🔤 폰트 · 눈누 상업·BI/CI 가능 재확인→
    눈누(noonnu.cc · 상업용 무료 한글 폰트 저장소)에서 '상업 이용 가능·BI(Brand Identity · 브랜드 정체성)/CI(Corporate Identity · 기업 정체성) 사용 가능' 필터로 다시 확인해 주세요.
  • 03🖼️ 이미지 · 의심 포인트는 참고 출처에 기록→
    2강에서 생성한 Gemini(제미나이 · 구글의 대화형 AI) 이미지 또는 무료 이미지를 씁니다. 의심 포인트는 매뉴얼 마지막 페이지 '참고 출처'에 적어 두세요.

패키징 중 검수 역할 담당자가 재점검할 3항목입니다.

이름은 KIPRIS 3건 재검색.

폰트는 눈누 라이선스 필터 '상업 이용 가능·BI/CI 사용 가능' 재확인.

이미지는 2강 Gemini 생성 이미지 또는 무료 이미지만 사용해요.

의심 포인트는 매뉴얼 마지막 페이지의 참고 출처에 기록합니다.

법률 참고로 말씀드리면, KIPRIS 검색과 폰트 라이선스 확인은 참고용이며 최종 법률 자문을 대체하지 않습니다. 상용화 단계에서는 변리사 상담을 권장해요.
