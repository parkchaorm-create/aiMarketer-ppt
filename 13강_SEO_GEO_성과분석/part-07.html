14강 · 콘텐츠 자동 생성 에이전트

  • 01 🤖 14강 주제 · 팀 프로젝트 →
    팀 단위로 콘텐츠 자동 생성 에이전트(반복 작업을 스스로 돌리는 AI 워크플로우)를 설계·실행합니다. 팀별 목적 맞춤.
  • 02 📑 재료 · 오늘 "3개월 액션 플랜" →
    오늘 만드신 3개월 액션 플랜을 재료로 "어떤 콘텐츠를 새로 만들지" 판단. 슬랙에서 그대로 불러옵니다.
  • 03 👥 팀 편성 · 같은 클라이언트끼리 →
    A/B/C/D 같은 가상 클라이언트를 선택한 수강생끼리 팀. 4강 팀 편성 방식과 동일. 별도 준비물 없습니다.

13강 전체 · 네 줄 정리

  • 01 📚 이론 · SEO·GEO·GSC·GA4 →
    SEO vs GEO 관계. GSC 5대 지표 + GA4 참여율·이탈률 공식 정의까지 판단 기준 확보.
  • 02 🔧 설계 · 입력·처리·출력 3단 →
    CSV+URL → Claude Code+WebFetch → Markdown. 매달 반복되는 작업을 재현 가능한 프로시저로 고정.
  • 03 📄 실습 · 리포트 2장 완성 →
    실습 A = URL 1개 진단 리포트. 실습 B = URL 3개 합산 리포트 + GEO 매트릭스 + 3개월 플랜.
  • 04 🛡️ 할루시네이션 · 사람 확인 기본 →
    AI 출력은 초안 전제. 숫자·문장은 발송 전 사람이 1회 확인. 자동화에서 생략 금지되는 단계입니다.
