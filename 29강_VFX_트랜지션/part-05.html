이미지 2장이, 연결 영상이 될까요?

  • 01 🎯 목표 · 3컷 중 가운데 →
    실습 A의 목표는 이미지 2장을 이어 3컷 시퀀스 중 가운데 전환 영상 1편을 남기는 것. 양 끝 컷은 여러분이 갖고 계세요.

    상세 단계는 노션 `실습_튜토리얼/실습A_트랜지션.md`에 있습니다. 이 슬라이드는 흐름 안내, 노션은 단계별 매뉴얼이에요.

03 / 07

준비부터 업로드까지

  • 01 🖼️ ① · 이미지 2장 준비 →
    Part 03에서 정리한 4조건을 맞춘 Start/End Frame 2장. 28강 영상에서 프레임 캡처가 가장 편합니다. Windows는 Win+Shift+S.
  • 02 🔗 ② · higgsfield.ai/app/transitions →
    힉스필드 Transitions 앱으로 직접 진입. 로그인된 상태면 바로 업로드 화면이 뜹니다. 왼쪽·오른쪽 업로드 박스가 보여요.
  • 03 ⬆️ ③ · 업로드 + 프리셋 선택 →
    왼쪽 Start Frame, 오른쪽 End Frame. 하단에서 Morph 또는 Seamless 클릭. 4조건 맞은 쌍이면 Seamless가 무난합니다.
  • 04 ⚡ ④⑤ · Generate + 슬랙 업로드 →
    Generate 클릭 → 20~60초 대기 → 다운로드. `#수업실습`에 `[29강][태그] 트랜지션_설명.mp4`로 업로드하면 실습 A 완료입니다.

04 / 07

A·B·C·D · Start/End 예시

  • 01 🌷 A 푸르트 · 꽃봉오리→꽃 →
    Start: 꽃봉오리 클로즈업, End: 활짝 핀 같은 꽃. Preset: Morph. 28강 영상 첫·끝 프레임을 그대로 쓰면 가장 쉽습니다.
  • 02 🍽️ B 슬로우템포 · 빈 접시→음식 →
    Start: 빈 브런치 접시(아침 햇살 측광), End: 음식 담긴 같은 접시(같은 조명). Preset: Seamless. 조명 일관성이 포인트예요.
  • 03 🏺 C 손결 · 흙덩이→그릇 →
    Start: 물레 위 흙덩이, End: 성형 중인 그릇(같은 작업대·같은 조명). Preset: Seamless. 작업대가 안 바뀌어야 자연스러워요.
  • 04 📖 D 링크드마인드 · 닫힌 책→펼친 책 →
    Start: 닫힌 책, End: 펼쳐진 같은 책(같은 책상·같은 조명). Preset: Morph. 책 위치가 같으면 모프가 특히 깔끔합니다.

05 / 07

Part 05 · 세 줄 정리

  • 01 🎞️ 2장 + 프리셋 = 1편 →
    이미지 2장에 트랜지션 프리셋 1개를 걸면 1~1.5초 전환 영상이 출력됩니다. 3컷 시퀀스의 가운데가 채워지는 구조예요.
  • 02 🥇 28강 캡처 · 가장 편한 1순위 →
    Start/End Frame 조달 1순위는 28강 카메라 워킹 영상의 첫·끝 프레임. 4조건이 자동으로 맞아 떨어집니다.
  • 03 ♻️ 재시도 · 1~2회 이내 →
    첫 결과가 기대와 달라도 괜찮습니다. 1~2회까지만 재시도. 완벽보다 경험 · 30강 팀 프로젝트의 재료라는 점 기억해 주세요.

이제 · 한 장면을 비틀까요?
