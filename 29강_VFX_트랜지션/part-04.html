03 / 07

로그인에서 Generate까지

  • 01 🔑 ①② · 로그인 + 앱 진입 →
    힉스필드 로그인 후 상단 메뉴의 Transitions 클릭. 경로는 /app/transitions. 창이 Start/End 업로드 박스 2개로 나뉘어 있어요.
  • 02 ⬆️ ③④ · Start + End 업로드 →
    왼쪽 박스에 Start Frame 1장, 오른쪽 박스에 End Frame 1장. Part 03에서 정리한 4조건이 맞는 쌍인지 확인 후 올립니다.
  • 03 🎛️ ⑤ · 프리셋 선택 →
    하단 또는 우측 패널에서 Morph 또는 Seamless 클릭. 클릭하면 미리보기 썸네일이 바뀌어 선택 상태가 보입니다.
  • 04 ⚡ ⑥ · Generate + 대기 →
    Generate 클릭 후 20~60초 대기. 결과 영상이 자동 재생되면 다운로드 버튼으로 저장. 크레딧(생성 횟수 단위)은 1회당 1~2 정도 소모됩니다.

📝 강사 노트
💬 그대로 읽으세요
1. Effects 앱 진입 · 상단 메뉴 "Effects" 또는 `/effects` 경로 2. 라이브러리 스크롤 · 50+ 이펙트 목록 확인[^T1-3] 3. 오늘 사용할 2종 미리 보기: 3D Rotation(3D 회전) · Glowshift(글로우시프트 · 빛 번짐 변화) 4. 이펙트 카드 클릭 → 상세 페이지에서 샘플 영상 확인 · `Use this effect(이 이펙트 사용하기)` 버튼 위치 확인

3D Rotation: 피사체가 입체 공간에서 회전하는 효과. 제품 샷·로고·인물 정면 컷 모두에 어울리는 가장 무난한 시작점.

Glowshift: 화면에 부드러운 빛이 번지며 색감이 바뀌는 효과. 꽃·음식·감성 제품 컷과 잘 맞음.

50여 개 중 · 오늘 2종

  • 01 📚 라이브러리 · 50+ 이펙트 →
    Effects 앱에 들어가면 50여 개 이펙트 카드가 스크롤로 펼쳐집니다. 카드 클릭 시 상세 페이지에서 샘플 영상을 볼 수 있어요.
  • 02 🌀 3D Rotation · 입체 회전 →
    피사체가 입체 공간에서 회전하는 효과. 제품 샷·로고·인물 정면 컷에 모두 어울리는 가장 무난한 시작점입니다.
  • 03 ✨ Glowshift · 빛 번짐 →
    부드러운 빛이 번지며 색감이 바뀌는 효과. 꽃·음식·감성 제품 컷과 잘 맞습니다. 원본이 어두우면 결과도 어두워져요.
  • 04 🛡️ 실패 4종 · 미리 대비 →
    중간 이상(4조건 어긋남)·과격(프리셋 특성)·Generate 비활성(크레딧/세션)·대기 2분 초과(서버 지연). 네 가지 경우와 대응을 미리 안내합니다.
CONCEPT · 02
📝 강사 노트
💬 그대로 읽으세요
여유되시는 수강생은 Datamosh(데이터모시 · 디지털 글리치처럼 화면이 깨지며 변형되는 효과)[^T1-3]를 추가로 시도하셔도 좋습니다. 단, Datamosh는 결과가 "예측 불가한 변형"이라 성공률이 낮습니다. 기본 2종을 먼저 확보한 뒤 남는 크레딧으로 도전해 주세요.

Transitions 결과 중간에 이상한 이미지가 낀다 → Start/End Frame 4조건(색감·조명·위치·스타일) 중 어긋난 요소가 있다는 신호. 이미지 재선정.

Part 04 · 세 줄 정리

  • 01 🧭 새 메뉴 2개 · 위치 확인 →
    Transitions(두 장면 잇기)와 Effects(한 장면 비틀기). 상단 메뉴에서 두 앱 위치를 눈으로 확인하셨습니다.
  • 02 🔘 핵심 버튼 3 · Start·End·Generate →
    Transitions의 핵심 버튼은 Start Frame 업로드·End Frame 업로드·Generate 세 개. 나머지는 보조입니다.
  • 03 💡 실패 전제 · 재시도 1~2회 →
    AI는 확률적 결과물. 완벽 아닌 "계정에 남기기"가 합격선. 재시도 1~2회 이내로 크레딧을 지킵니다.
