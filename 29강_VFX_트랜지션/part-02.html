Transition과 VFX · 한 문장 구분

  • 01 🎞️ 카드 1 · Transition 개념 →
    Transition(트랜지션 · 화면 전환)은 두 장면을 자연스럽게 잇는 기술. Start Frame + End Frame 2장 입력이 핵심입니다.
  • 02 ✨ 카드 2 · VFX 개념 →
    VFX(Visual Effects · 시각 효과)는 한 장면에 효과를 덧입히는 기술. 이미지 1장 + 이펙트 프리셋 1개 구조입니다.
  • 03 🧭 메뉴 찾기 · 한 줄 암기 →
    "두 장면 이을 땐 Transitions, 한 장면 비틀 땐 Effects." 힉스필드 메뉴 두 개를 구분하는 외우기 쉬운 한 줄 기준입니다.

02 / 07

'전환'과 '효과', 같은 말 아닌가요?

  • 01 🤔 일상어 · 비슷해 보이는 두 단어 →
    "트랜지션 = 효과 아닌가?" 많이 헷갈려하시는 지점입니다. 일상어에선 두 단어가 겹쳐 쓰이니 자연스러운 혼동입니다.
  • 02 🎬 현장어 · 아주 다른 기능 →
    영상 제작 현장에서는 두 단어가 아주 다른 기능을 가리킵니다. 입력도, 결과도, 메뉴도 다릅니다. 이걸 구분해 두면 많은 게 편해집니다.
  • 03 🗺️ 정리하면 · UI 헤맴 없음 →
    이 차이를 정리해 두시면, 오늘 힉스필드 UI에서 어느 메뉴를 눌러야 할지 고민이 없어집니다. 첫 단추가 맞아지면 실습이 빨라져요.

CONCEPT · 카드 1 · Transition
03 / 07

두 장면을 잇는 기술

  • 01 🎞️ 입력 · 이미지 2장 →
    Start Frame(시작 프레임 · 첫 장면 이미지)과 End Frame(끝 프레임 · 끝 장면 이미지) 두 장이 필수 입력입니다. 이게 전제예요.
  • 02 ⚙️ 결과 · 1~1.5초 전환 영상 →
    두 장면을 잇는 짧은 영상이 출력됩니다. Cut(컷 · 뚝 끊기)으로 끊는 대신, Morph(모프 · 변형) 같은 방식으로 매끄럽게 넘어가요.
  • 03 🌷 예시 · 꽃봉오리 → 활짝 핀 꽃 →
    A 푸르트 기준으로 꽃봉오리 사진에서 활짝 핀 꽃 사진으로 Morph. 1초 남짓 사이에 자연스럽게 피어나는 애니메이션이 생성됩니다.
  • 04 📍 메뉴 · Transitions 앱 →
    힉스필드 상단·좌측 메뉴의 Transitions 전용 앱에서 작업합니다. Dissolve(디졸브 · 서서히 섞이는 전환)류 프리셋도 이 안에 있어요.

CONCEPT · 카드 2 · VFX
04 / 07

한 장면을 비트는 기술

  • 01 🖼️ 입력 · 이미지 1장 + 프리셋 →
    VFX는 이미 있는 이미지 1장과 이펙트 프리셋 하나를 고르는 구조입니다. "이 사진을 이런 식으로 변형해 달라"는 지시예요.
  • 02 🎁 결과 · 5초 특수효과 영상 →
    같은 꽃다발 사진에 3D Rotation을 걸면 공간에서 회전하는 입체 영상, Glowshift를 걸면 빛이 번지는 감성 영상으로 태어납니다.
  • 03 🎯 용도 · 시선 잡기 · 단독 강조 →
    VFX는 단독 컷을 강조하거나 드라마틱 연출을 만들 때 씁니다. 같은 이미지라도 프리셋에 따라 완전히 다른 인상이 나옵니다.
  • 04 📍 메뉴 · Effects 라이브러리 →
    힉스필드 Effects 라이브러리에 50여 개 이펙트가 있습니다. 오늘은 기본 2종(3D Rotation · Glowshift)을 사용합니다.

RECAP · 복습
05 / 07

Part 02 · 세 줄 정리

  • 01 🎞️ Transition · 두 장면 잇기 →
    Start Frame + End Frame 2장 입력, Transitions 메뉴. 연속된 상태 변화·브랜드 스토리의 장면 전환에 씁니다.
  • 02 ✨ VFX · 한 장면 비틀기 →
    이미지 1장 + 이펙트 프리셋, Effects 메뉴. 시선 잡기·드라마틱 연출·단독 컷 강조에 씁니다.
  • 03 🗒️ 오늘 합계 · 3편 →
    실습 A에서 트랜지션 1편, 실습 B에서 VFX 2편. 합쳐서 오늘 3편의 새 영상이 계정에 쌓입니다.

BRIDGE · 다음 파트
06 / 07

이미지 2장, 어떻게 고를까요?

  • 01 🖼️ 2장 품질 · 결과 좌우 →
    트랜지션은 이미지 2장을 무엇으로 고르느냐에 따라 결과 품질이 크게 달라집니다. 아무 2장 넣어도 되긴 하지만요.
  • 02 📐 Part 03 · 좋은 쌍의 4조건 →
    색감 일관성·조명 방향·피사체 위치·스타일 통일. 네 가지가 맞을수록 AI가 "중간"을 상상하기 쉬워집니다.
  • 03 🧺 3가지 조달 경로 · 28강 활용 →
    28강 영상의 첫·끝 프레임이 1순위, 클라이언트 사례집이 2순위, 수업 중 생성이 3순위. Part 03에서 정리합니다.
