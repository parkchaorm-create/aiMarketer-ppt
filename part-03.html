Part 03 · 22분 뒤 당신의 변화

  • 01⌨️ 터미널 5개 동작 · 까만 화면이 편해집니다→
    pwd, dir, cd, mkdir, claude — 이 다섯 개만 익히면 백스테이지를 마음껏 누빌 수 있어요. 오늘 파트에서 직접 따라 치면서 배웁니다.
  • 02🛡️ Plan Mode · 삽질 80%를 사전에 제거합니다→
    AI한테 바로 실행시키기 전에 계획을 먼저 보여달라고 하는 기능이에요. 리허설 한 번이 삽질 열 번을 막아줍니다.
  • 03⏪ /rewind · 실수해도 한 방에 되돌립니다→
    리허설 없이 공연했다가 망했어도 괜찮아요. /rewind 한 줄이면 최근 변경사항을 취소하고 다시 시작할 수 있거든요.

  • 01😱 터미널 공포 · 열자마자 닫아본 적 있죠?→
    저도 처음에 터미널 열었다가 아무것도 안 하고 그냥 닫아버렸어요. 근데 지금은 이게 없으면 일을 못 합니다.
  • 02🎭 정체 · 무대 뒤 통로(백스테이지)일 뿐→
    마우스로 폴더 클릭하는 걸 글로 하는 것뿐이에요. 관객은 몰라도 되지만, 연주자는 이 통로로 무대에 올라가야 하거든요.
  • 03🔑 동작 5개 · 배우면 가장 편한 길이 됩니다→
    딱 다섯 개만 알면 이 통로가 제일 편한 지름길이 돼요. 오늘 파트 끝나면 터미널 공포증 완치 보장합니다!

02 / 09

터미널 = 백스테이지 직행 통로

  • 01🖱️ 마우스 · 우클릭 3단계가 필요→
    마우스로 폴더 만들 때 우클릭, 새 폴더, 이름 입력 세 단계가 필요하죠. 터미널에서는 mkdir marketing 한 줄로 끝이에요.
  • 02⌨️ 터미널 · mkdir marketing 한 줄이면 끝→
    터미널이 느리고 불편해 보여도, 익숙해지면 마우스보다 훨씬 빠르거든요. 글로 명령하는 게 얼마나 강력한지 직접 느껴보세요.
  • 03🎵 Claude Code · 이 통로에서만 실행 가능→
    오케스트라 음향 장비가 무대 뒤 통로에 있으니까요. 터미널 없이는 Claude Code를 켤 수 없어요. 이게 터미널을 배워야 하는 진짜 이유입니다.

03 / 09

알면 편한 꿀팁 2가지

  • 01⌨️ Tab 키 · 자동완성으로 오타 제로→
    폴더 이름을 다 치지 않아도 Tab 키 한 번이면 자동완성됩니다. 오타 걱정 없이 빠르게 입력할 수 있어요.
  • 02⬆️ 위 화살표 · 이전 명령어 즉시 재호출→
    키보드 위쪽 화살표를 누르면 이전 명령어가 다시 나타납니다. 같은 명령어를 반복할 때 타이핑할 필요가 없어요.
  • 03🚀 결과 · 타이핑 반으로 줄고 속도 두 배→
    Tab 자동완성과 위 화살표, 이 두 가지만 쓰면 타이핑량이 절반으로 줄고 작업 속도가 확 올라갑니다.

Plan Mode · 왜 필요한가?

  • 01💥 리허설 없는 본공연 · 파일 20개 엉뚱 생성→
    계획 없이 실행 → 파일 20개 엉뚱하게 생성 → 수습에 시간 낭비
  • 02🎻 합주 불일치 · 각자 다른 곡을 연주→
    바이올린은 모차르트, 첼로는 베토벤 -- 리허설 없으면 합주가 엉망이 됩니다.
  • 03🛡️ 해결책 · 리허설 한 번에 삽질 80% 제거→
    Plan Mode = 리허설 / 승인 후 실행 → 삽질 80% 제거

Edit Mode vs Plan Mode · 두 가지 모드

  • 01⚡ Edit Mode · 기본값 · 바로 파일 생성/수정→
    Shift+Tab 즉시 전환 / 단순 수정·빠른 실행에 최적
  • 02🎭 Plan Mode · 계획 먼저 · OK하면 그때 실행→
    계획 텍스트 먼저 출력 → 승인 후 실행 / 새 기능·복잡 작업
  • 03📋 선택 기준 · 복잡하면 Plan, 간단하면 Edit→
    헷갈리면 Plan / 잘못돼도 /rewind 복구 가능

규칙 하나 · 처음 만드는 거면 리허설!

  • 01🎭 Plan 필수 · 새 파일, 복잡한 설계일 때→
    새 파일·여러 파일 수정 → Plan 시작 / 실수 방지
  • 02⚡ Edit OK · 단순 수정, 간단 질문일 때→
    한 줄 수정·단순 질문 → Edit 충분 / Shift+Tab 전환
  • 03⏪ /rewind · 망해도 되돌리기 가능→
    /rewind로 되감기, Esc 두 번이면 긴급 중단. 실수해도 언제든 복구 가능합니다.

Part 03 · 요약 세 줄

  • 01⌨️ 터미널 5개 · pwd / dir / cd / mkdir / claude면 충분→
    위치 확인, 목록 보기, 이동, 폴더 만들기, Claude 실행. 이 5개면 오늘 영상 전부 따라하실 수 있습니다.
  • 02🎭 새 작업 = Plan Mode · "계획 먼저 보여줘" 한마디→
    새로 만드는 작업은 반드시 Plan Mode로 리허설 먼저! "계획 먼저 보여줘" 한마디면 삽질 80%가 사라집니다.
  • 03⏪ 실수 = /rewind · 겁먹지 마세요, 되돌립니다→
    실수해도 /rewind로 되돌리기 가능하고, Esc 두 번이면 긴급 중단됩니다. 겁먹지 마세요!
