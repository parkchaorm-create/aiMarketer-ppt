실습 B · 7 단계 흐름

  • 01 📋 7 단계 · Canva → PDF →
    접속 → 템플릿 검색 → 선택 → 표지 커스텀 → 본문 교체 → 브랜드 반영 → PDF 다운로드 → 슬랙 업로드.
  • 02 🎨 무료 템플릿 · 3 기준 선택 →
    왕관(Pro) 마크 없는 무료(Free) · 7~15페이지 이내 · 비즈니스 클린 스타일 셋 다 만족해야 안전합니다.
  • 03 🎯 합격선 · 4 페이지 + PDF →
    시간 부족하면 4페이지(표지·요약·가격·다음 단계)만 채워도 PDF 다운로드까지 가면 합격선. 나머지 자율 확장.

02 / 07

디자인 · 완벽해야 할까요?

  • 01 🚪 메일이 문 · 제안서가 테이블 →
    콜드메일이 문을 여는 도구라면, PDF 제안서는 문을 지나 테이블에 앉는 도구입니다. "한번 뵐까요" 답장에 곧바로 보낼 한 장.
  • 02 🎨 템플릿 · 90% 완성된 뼈대 →
    Canva 템플릿은 이미 90% 완성된 뼈대예요. 우리는 텍스트 교체 + 로고 삽입 + PDF 다운로드만 하면 됩니다.
  • 03 🎯 오늘 목표 · PDF까지 →
    고급 레이아웃·이미지 매칭은 수강 후 자율 확장 영역. 오늘은 PDF Standard로 다운로드해 슬랙에 올리는 것까지가 목표예요.

B · 슬로우템포 · 예시

  • 01 📘 표지 · 에이전시·클라·날짜 →
    "슬로우템포를 위한 SEO 블로그 제안 · 작은달력 · 2026-04". 4개 정보(에이전시·대상·제목·날짜)만 명확히.
  • 02 📝 요약·과제 · 한 문단씩 →
    요약(3줄) · 과제(검색 노출 부족 · SNS 한계 · 지역 유입 불투명 3개). 진단을 짧게 · 자세하게.
  • 03 📦 전략·범위·가격 · 32강 인용 →
    전략(SEO+GBP+리포트) · 범위(블로그 4편/월·GBP 1회·리포트 1회) · 가격(32강 패키지 A · "사례별 상이" 각주 필수).
  • 04 🚀 다음 단계 · 3 단계로 →
    "30분 화상 인터뷰 → 1주 내 맞춤 제안서 업데이트 → 계약 시 2주 내 착수." 명확한 시간선이 신뢰를 만듭니다.

04 / 07

검색 키워드 · PDF Standard

  • 01 🔎 검색어 · 영문 3 개 추천 →
    "pitch deck minimal" · "marketing proposal" · "business proposal modern" 셋 중 하나로 시작. 영어 키워드가 결과 더 많아요.
  • 02 ✅ 선택 기준 · 3 가지 동시 만족 →
    왕관(Pro) 마크 없는 무료 · 7~15페이지 이내 · 비즈니스 클린 스타일. 화려한 일러스트·원색 과다 템플릿은 패스.
  • 03 📥 다운로드 · PDF Standard →
    Share(공유) → Download(다운로드) → PDF Standard. 표준 품질이 메일 첨부·클라우드 공유에 충분합니다. PDF Print는 인쇄용.
  • 04 ⚠️ 주의 · Pro 요소·개인정보·과장 →
    드래그 시 왕관 보이면 무료 대체. 실제 기업 담당자 실명·연락처 X. "업계 최고"·"100% 보장" 자동 제거.

Part 06 · 세 줄 정리

  • 01 🎨 Canva · 무료 템플릿 7~15p →
