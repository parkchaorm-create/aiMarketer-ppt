콜드메일이 문을 여는 도구라면, 제안서 PDF는 문을 지나 테이블에 앉는 도구입니다. 고객이 메일 답장으로 "한번 뵐까요"라고 말했을 때, 곧바로 보낼 수 있는 표지 + 본문 7~8페이지를 오늘 완성합니다.

실습 B는 7단계로 나눠 진행합니다. 세부 단계는 노션 튜토리얼(`실습_튜토리얼/실습B_제안서PDF.md`)에 화면 캡처와 함께 정리돼 있어요.

1. Canva 접속 + 템플릿 검색 — "pitch deck" 또는 "proposal" 카테고리 2. 템플릿 선택 — 무료(Free) 마크가 붙은 템플릿 중 7~8페이지 이내로 3. 표지 커스터마이징 — 에이전시명·클라이언트명·제목·날짜 4. 본문 페이지 교체 — 요약 · 과제 · 전략 · 범위 · 가격 · 다음 단계 5. 브랜드 요소 반영 — 4강 팀 매뉴얼의 로고·컬러·폰트 적용 6. PDF 다운로드 — Share(공유) → Download(다운로드) → PDF Standard 7. 슬랙 업로드 — `#수업실습`에 `[33강][A/B/C/D] 제안서PDF` 업로드

Marketing → Emails → Templates

  • 01 📂 경로 · 3 단계 메뉴 →
    GHL 좌측 Marketing(마케팅) → Emails(이메일) → Templates(템플릿) 클릭 → New Template 버튼.
  • 02 🏷️ 이름 · 영문 컨벤션 권장 →
    outreach_T1_first_touch · outreach_T2_followup · outreach_T3_final 같은 영문 스네이크 케이스. 이후 워크플로우에서 검색 쉬워요.
  • 03 🏗️ Merge Tag · 두 가지만 →
    Merge Tag(머지 태그 · 변수 자리) 두 개만 사용. {{contact.first_name}} {{contact.company_name}} . 다른 변수 X.
  • 04 🐛 자주 막힘 · 4 가지 →
    영어 답변(언어 지정 누락) · 본문 길이 초과 · Merge Tag 누락 · 저장 버튼 회색(필수 필드). 노션 튜토리얼에 1차 조치 표 있어요.

1. 강사가 전면 화면에서 표지 + 요약 페이지 2장을 시연 — 템플릿 검색 → 선택 → 텍스트 교체 → 로고 삽입. 2. 시연 후 수강생은 각자 자리에서 본인 클라이언트용 제안서를 커스터마이징. 우선순위는 표지 → 요약 → 가격 → 다음 단계 순서. 시간이 부족하면 과제·전략·범위 페이지는 짧게. 3. 완성 후 PDF 다운로드 → 슬랙 `#수업실습`에 `[33강][A/B/C/D] 제안서PDF` 업로드. 파일이 크면 스크린샷 4~6장 몽타주로 대체 가능합니다. 4. 막히는 분은 `#강사-QnA` 채널 또는 손들기로 강사·운영진 호출.
