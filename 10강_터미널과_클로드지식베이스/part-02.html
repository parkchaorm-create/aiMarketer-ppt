터미널 · 왜 지금 돌아오는가

📝 강사 노트
💬 그대로 읽으세요
최근 1~2년 사이에 "AI를 잘 쓰는 사람은 결국 터미널로 돌아온다"라는 이야기가 꾸준히 들립니다. 이유는 단순합니다. AI 에이전트가 여러 도구를 순서대로 엮어 작업하려면, 클릭 한 번으로는 부족해서예요. 마우스는 한 창에 한 작업에 최적화되어 있는 반면, 터미널은 여러 명령을 한 줄씩 · 순서대로 · 기록으로 남겨 돌리는 데 강합니다.

그리고 클로드 코드(Claude Code)는 처음부터 터미널 안에서 사는 AI 에이전트로 설계됐습니다[^1]. 웹 브라우저가 아니라 터미널 창이 집이죠. 오늘 우리는 그 집의 초인종을 누르는 방법을 배우는 겁니다.

    터미널은 여러 명령을 한 줄씩 · 순서대로 · 기록으로 남겨 돌리는 데 강합니다.
  • 03 🏠 Claude Code · 집이 터미널 →

03 / 07

터미널은 창 · CLI는 대화 방식

  • 01 🪟 터미널 · 글자를 주고받는 창 →
    검은 배경에 글자를 입력받고 결과를 글자로 뿌려 주는 창입니다. 윈도우 PowerShell · 맥 Terminal 모두 포함.
  • 02 💬 CLI · 대화 방식 자체 →
    CLI(Command-Line Interface)는 "명령어 한 줄 → 결과 한 줄"의 구조 자체를 가리키는 말입니다.
  • 03 🖼️ GUI · 아이콘 클릭 방식 →
    GUI(Graphical User Interface)는 우리가 지금까지 써 온 아이콘 · 버튼 · 창을 클릭하는 방식입니다.
  • 04 🔗 터미널과 CLI · 늘 함께 →
    쉽게 말하면 · 터미널은 "창" · CLI는 "대화 방식". 두 단어는 항상 붙어 다닙니다.
CONCEPT · 01
📝 강사 노트
💬 그대로 읽으세요
업계 관점에서 본 이유를 세 가지로 정리할 수 있습니다.

1. 재현성(Reproducibility): 마우스 클릭은 기록이 남기 어렵지만, CLI는 명령어 그 자체가 곧 기록입니다. 같은 명령을 다시 붙여 넣으면 같은 결과가 나옵니다. 2. 조합성(Composability): 한 명령의 결과를 바로 다음 명령에 넘길 수 있습니다. AI 에이전트가 "파일 목록 → 내용 읽기 → 요약 → 저장"처럼 여러 작업을 자동으로 엮는 기본 조건입니다. 3. AI 에이전트 친화성: 최근의 에이전트형 개발 도구들(클로드 코드·제미나이 CLI·Codex CLI 등)은 모두 터미널을 본거지로 삼고 있습니다[^1][^2]. AI가 컴퓨터에 "일을 시키기"에 가장 적합한 통로이기 때문입니다.

04 / 07

재현성 · 조합성 · AI 친화성

  • 01 🔂 재현성 · 명령 = 기록 →
    재현성(Reproducibility · 같은 입력이면 같은 결과). CLI는 명령어 그 자체가 곧 기록이라 다시 붙이면 똑같이 돌아갑니다.
  • 02 🧩 조합성 · 결과 → 다음 명령 →
    조합성(Composability · 명령을 엮어 쓰기). 한 명령의 결과를 바로 다음 명령에 넘겨 여러 작업을 자동으로 엮습니다.
  • 03 🤖 AI 친화 · 에이전트의 본거지 →
    Claude Code · Gemini CLI · Codex CLI 등 최근 AI 에이전트형 도구는 모두 터미널을 본거지로 삼고 있습니다.

05 / 07

pwd · ls · cd · mkdir · 딱 4개

  • 01 📍 pwd · 지금 나 어디? →
    pwd(print working directory · 현재 작업 폴더 출력). 지금 내가 서 있는 폴더 경로를 글자로 알려 줍니다.
  • 02 👀 ls · 주변에 뭐가 있지? →
    ls(list · 목록 보기). 지금 폴더 안에 어떤 파일·폴더가 있는지 나열합니다. 윈도우는 dir도 가능.
  • 03 🚶 cd · 저쪽으로 걸어가자 →
    cd {폴더명}(change directory · 폴더 이동). 지정한 폴더 안으로 이동합니다. cd ..는 한 칸 위로.
  • 04 🧱 mkdir · 자리 하나 새로 깔자 →
    mkdir {폴더명}(make directory · 폴더 만들기). 새 폴더를 만듭니다. 같은 이름 두 번은 안전하게 무시됩니다.
