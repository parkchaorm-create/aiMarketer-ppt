실습 A · 다섯 단계

  • 01 📂 복제 · 11강 스킬 폴더 →
    터미널에서 `cp -r` (맥) 또는 `Copy-Item -Recurse`(Win) 명령으로 social-media 폴더를 리퍼포징 폴더로 복제합니다.
  • 02 ✏️ 3구역 · 프론트·목적·본문 →
    프론트매터 수정 · 목적 블록 교체 · 지시문 본문을 세 채널 섹션 3개로 교체합니다.
  • 03 ✅ 저장 · 자동완성 확인 →
    `claude` 실행 후 `/` 키를 눌러 자동완성 목록에 `/리퍼포징`이 나타나면 실습 A 성공입니다.
OVERVIEW
📝 강사 노트
💬 그대로 읽으세요
1. 기존 스킬 복제: 터미널에서 `cp -r .claude/skills/social-media .claude/skills/리퍼포징` 입력. (Windows PowerShell은 `Copy-Item -Recurse`를 씁니다. 노션 튜토리얼에 OS별 명령이 병기되어 있어요.) 2. 프론트매터 수정: 복제된 `SKILL.md`를 열어, `name: social-media`를 `name: 리퍼포징`으로 바꾸고, `description`을 "블로그 원문을 인스타·스레드·블로그 요약 세 형식으로 변환"으로 바꿉니다. `disable-model-invocation: true` 한 줄을 추가합니다. 3. 목적 블록 수정: 본문 서두의 "이 스킬은 ..." 문장을 리퍼포징 용도로 바꿉니다. 노션 튜토리얼에 붙여넣기용 템플릿이 준비되어 있습니다. 4. 지시문 본문 교체: 본문 중반의 "출력 형식" 섹션을 세 채널 섹션 3개로 교체합니다. 각 섹션의 길이·톤·해시태그 규칙은 Part 03 카드 1의 표 그대로 사용하세요. 5. 저장 후 확인: 터미널에서 `claude` 실행 → `/` 키를 눌러 자동완성 목록에 `/리퍼포징`이 나타나는지 확인합니다.

도구 선택 · 인코딩 · 업로드

  • 01 🖥️ 도구 · 안티그래비티 권장 →
    11강에서 쓰시던 Antigravity IDE 또는 익숙한 에디터. Windows 메모장은 한국어 인코딩 문제로 피합니다.
  • 02 💾 인코딩 · UTF-8로 저장 →
    파일은 반드시 UTF-8(유니코드 · 한국어를 안전히 담는 표준)로 저장합니다. 메모장은 기본값이 달라 깨질 수 있습니다.
  • 03 📤 슬랙 · 수업실습 업로드 →
    완성된 파일을 복사해 `#수업실습`에 업로드합니다. 파일명 예시: `[12강][A] SKILL_닉네임.md`.
