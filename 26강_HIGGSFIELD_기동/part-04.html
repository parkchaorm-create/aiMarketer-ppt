    프롬프트 붙여넣기 · Generate(생성) 버튼 · 20~60초 대기 · 결과 재생·다운로드까지 한 번 돌려 봅니다.

영어 UI · 겁먹으셨나요?

  • 01 🌐 힉스필드 · 영어 기본 UI →
    힉스필드는 한국어 UI 정식 지원이 아직 없습니다. 2026-04 기준 영어가 기본입니다.
  • 02 🔄 크롬 자동 번역 · 해결 →
    오른쪽 마우스 → "한국어로 번역" 한 번이면 메뉴가 한국어로 표시됩니다. 번역이 어색하면 원문 영어를 참고합니다.
  • 03 📖 오늘 수업 · 영어·한국어 병기 →
    주요 버튼명은 Sign up(가입) · Create(만들기) · Generate(생성) 식으로 원문+한국어를 함께 안내합니다.

03 / 07

접속 · 가입 · 대시보드 · 생성

  • 01 🌐 단계 1·2 · 접속과 가입 →
    higgsfield.ai 접속 후 Sign up 클릭. 이메일 또는 Google 계정 연결로 빠르게 가입합니다.
  • 02 📊 단계 3·4 · 대시보드·크레딧 →
    로그인 후 메인 대시보드. 우측 상단에 무료 크레딧 수 표시. 무료 플랜 일일 약 10 크레딧 수준입니다.
  • 03 🎨 단계 5 · Create 진입 →
    상단 네비게이션의 Video 또는 Create 메뉴. 입력창이 보이면 진입 성공입니다.
  • 04 ▶️ 단계 6·7 · Generate와 결과 →
    4요소 프롬프트 붙여넣고 Generate 클릭. 20~60초 후 썸네일 클릭 → 재생 → 다운로드.

04 / 07

흔한 막힘 5가지 · 미리 알기

  • 01 📧 가입 메일 · 늦게 올 수 있음 →
    인증 이메일이 안 오면 먼저 스팸함 확인. 그래도 없으면 #운영-QnA 채널에 바로 문의하세요.
  • 02 🔒 Generate · 회색 비활성 →
    버튼이 눌리지 않으면 크레딧 부족 또는 로그인 세션 만료. 새로고침 후 재로그인하시면 풀립니다.
  • 03 ⏳ 대기 · 1분 넘어가도 정상 →
    보통 20~60초. 1분 넘어가도 정상 범위. 2분 넘어가면 강사에게 알려 주세요.
  • 04 🎲 결과 · 기대와 다를 수 있음 →
    확률적 결과물이라 가장 흔한 상황. 프롬프트 주인공 부분을 구체화해 한 번 더 시도합니다.

Part 04 · 세 줄 정리

  • 01 🌐 영어 UI · 크롬 번역 해결 →
