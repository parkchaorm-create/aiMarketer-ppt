04 / 07

앞면 · 뒷면 · 두 덩어리

  • 01 🔖 앞면 · YAML frontmatter →
    YAML(야믈 · 파일 맨 위 `---`와 `---` 사이에 쓰는 설정 문법) frontmatter에 name·description 두 줄을 넣어 카드 앞면을 만듭니다.
  • 02 📝 뒷면 · 마크다운 본문 →
    `---` 아래의 마크다운 텍스트가 조리법입니다. "어떤 순서로·어떤 원칙으로·어떤 포맷으로" 자세히 적습니다.
  • 03 📂 저장 위치 · 세 가지 →
    개인(`~/.claude/skills/`) · 프로젝트(`.claude/skills/`) · 플러그인 세 곳. 오늘은 프로젝트 스킬로 만듭니다.

CLAUDE.md · Skill · MCP

  • 01 🏠 CLAUDE.md · 상주 지식 →
    프로젝트 전반 배경·규칙·용어집이 담긴 안내서. 프로젝트를 열 때 항상 읽힙니다.
  • 02 📇 Skill · 반복 작업 레시피 →
    특정 작업(캡션·리뷰·분석)의 순서와 원칙. 필요할 때만 불러옵니다. 오늘의 주인공.
  • 03 🔌 MCP · 외부 서비스 통로 →
    MCP(Model Context Protocol · 모델 컨텍스트 연결 규약). GHL·GitHub·슬랙 같은 외부 연결 규약. Phase 4에서.
