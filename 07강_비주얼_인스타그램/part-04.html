왜 도구 하나로는 안 될까요?

  • 01🖼️ 이미지 담당 · 그림 그리는 제미나이→
    제미나이(Gemini: 구글의 대화형 AI)는 카드뉴스용 이미지 다섯 장을 생성하는 전담입니다. 글은 잘 못 써도 그림은 잘 그립니다.
  • 02💬 글 담당 · 문구 다듬는 클로드→
    클로드(Claude: Anthropic의 대화형 AI)는 블로그 본문을 여섯 장 분량으로 요약하고 캡션(이미지 아래 설명 글)과 해시태그(# 기호로 주제를 분류하는 태그)까지 맡아 씁니다.
  • 03🧱 조립 담당 · 한 장으로 합치는 도구→
    미리캔버스(국내 웹 기반 디자인 도구)나 캔바(Canva: 글로벌 웹 기반 디자인 도구)는 이미지와 글자를 한 장으로 조립합니다. 각자 잘하는 일이 달라 셋을 묶어 씁니다.

제미나이 · Nano Banana 2

  • 01🍌 Nano Banana 2 · 기본 이미지 모델 (코드네임)→
    제미나이 앱의 이미지 생성은 2026-02 업데이트로 Nano Banana 2(공식명 Gemini 3.1 Flash Image) 모델을 쓰고 있습니다. 코드네임(Code Name · 내부 개발 단계 이름)이라 UI 표기나 버전이 바뀔 수 있습니다.
  • 02💸 무료 사용 · 로그인 필요→
    무료 사용자도 쓸 수 있지만 로그인 상태에서만 이미지 생성이 동작합니다.
  • 03⚡ 속도 · 빠른 기본 응답→
    기본 속도가 빠른 편이고 이미지 안의 한국어 글자 렌더링(Rendering: 문자를 이미지로 그려내는 처리)도 이전보다 정확해졌습니다.
  • 04🔁 오늘 · 다섯 번 호출 예정→
    오늘은 이 기본 이미지 생성을 다섯 번 호출해 이미지 다섯 장을 만듭니다.

클로드 · 두 가지 역할

미리캔버스 · 캔바

  • 01🇰🇷 미리캔버스 · 한국어 템플릿 풍부→
    미리캔버스는 한국어 템플릿과 글꼴이 풍부해 국내 이용자에게 익숙한 도구입니다.
  • 02🌐 캔바 · 글로벌 템플릿 많음→
    캔바는 글로벌 템플릿이 많아 영어 프로젝트나 해외 브랜드 참고에 유리합니다.
  • 03💳 무료 플랜 · 둘 다 카드뉴스 제작→
    두 도구 모두 무료 플랜에서 카드뉴스 제작이 가능하고 유료 요소는 쓰지 않아도 충분합니다.
  • 04🎛️ 자유 선택 · 특정 도구 강요 없음→
    오늘 강의는 두 도구 사용법을 특정하지 않고 진행하며 수강생이 편한 쪽을 선택하면 됩니다.

  • 011️⃣ 옆자리 · 30초 증상 확인→
    막히면 먼저 옆자리 팀원에게 같은 증상이 있는지 30초만 확인하시면 됩니다.

    그래도 막히면 슬랙 #강사-QnA 채널에 스크린샷 한 장과 한 줄 설명을 올리시면 됩니다.
  • 04🏢 보안 이슈 · 개인 기기 사용→
    회사 PC 보안 정책으로 접속이 막히면 개인 컴퓨터나 태블릿으로 진행하시거나 쉬는 시간에 강사에게 알려 주세요.
