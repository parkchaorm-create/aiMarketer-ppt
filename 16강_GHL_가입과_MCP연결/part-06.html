한 줄 명령어 · 조각 풀이

  • 01 🏷️ claude mcp add ghl · 이름 붙임 →
    Claude Code에 MCP 서버 추가. 별명 ghl은 내가 부를 이름. --transport http는 HTTP 방식 통신(GHL 요구 사항).
  • 02 🌐 --scope user · 어디서든 사용 →
    내 사용자 전체에서 사용 가능하게(프로젝트 폴더 상관없이). 프로젝트 전용으로 묶고 싶으면 --scope project.
  • 03 🔑 두 --header · 두 값 전달 →
    Authorization: Bearer PIT(인증 열쇠) · locationId: LOC(어느 서브계정인지). 따옴표 필수, 공백 혼입 금지.
  • 04 🧹 실행 후 · clear로 화면 지우기 →
    터미널에 PIT가 그대로 찍힐 수 있어요. 화면 공유 중이시면 clear(Mac) 또는 Ctrl+L로 터미널 한 번 지우세요.

Claude Code가 내부적으로 GHL MCP의 `contacts` 관련 도구를 호출하고, 숫자 한 개를 응답합니다. 예: `현재 연락처 수: 0명` (갓 만든 계정이라 0명이 정상).

3단계의 "숫자 응답"을 받으신 분은 오늘 수업 산출물 2번(Claude Code ↔ GHL MCP 통신 확인)을 완성하신 것입니다.

다음 두 가지를 슬랙 `#수업실습`에 올려 주세요.

1. `claude mcp list` 실행 결과 스크린샷 (PIT·Location ID가 보이지 않는 화면) 2. 3단계 자연어 테스트 응답 스크린샷 (숫자 한 개 포함)
