한 줄 명령어 · 조각 풀이

  • 01 🏷️ claude mcp add ghl · 이름 붙임 →
    Claude Code에 MCP 서버 추가. 별명 ghl은 내가 부를 이름. --transport http는 HTTP 방식 통신(GHL 요구 사항).
  • 02 🌐 --scope user · 어디서든 사용 →
    내 사용자 전체에서 사용 가능하게(프로젝트 폴더 상관없이). 프로젝트 전용으로 묶고 싶으면 --scope project.
  • 03 🔑 두 --header · 두 값 전달 →
    Authorization: Bearer PIT(인증 열쇠) · locationId: LOC(어느 서브계정인지). 따옴표 필수, 공백 혼입 금지.
  • 04 🧹 실행 후 · clear로 화면 지우기 →
    터미널에 PIT가 그대로 찍힐 수 있어요. 화면 공유 중이시면 clear(Mac) 또는 Ctrl+L로 터미널 한 번 지우세요.

검증 · 자연어 · 청소

  • 01 📋 1단계 · claude mcp list →
    ghl이 ✓ Connected 상태로 보이면 Claude Code가 서버 정보를 기억하고 있다는 뜻. 1단계 통과.
  • 02 🔍 2단계 · claude mcp get ghl →
    서버 이름·URL·transport·추가 시각 표시됨. 에러가 아니면 설정값이 올바르다는 증거. 2단계 통과.
  • 03 💬 3단계 · 자연어 질문 · 핵심 →
    "ghl 서버로 내 서브계정 연락처 수 세어서 숫자만 알려주세요." 숫자 한 개(보통 0)가 돌아오면 · 진짜 GHL과 · 말이 통한 겁니다.
  • 04 🧹 청소 · remove + delete + 메모장 →
    claude mcp remove ghl · GHL Settings에서 "16강-실습용" Delete · 메모장 PIT 줄 삭제. 세 곳 모두 비워야 완료.

Part 06 · 세 줄 정리

  • 01 🤝 연결 확인 · 숫자 1개로 증명 →
    3단계 검증 프로토콜(list·get·자연어) 모두 통과. "연락처 수: 0명" 같은 숫자 응답이 오늘의 공식 학습 산출물.
  • 02 📸 스크린샷 · 두 장 업로드 →
    #수업실습에 list 결과 + 자연어 응답 두 장. PIT·Bearer가 보이면 절대 올리지 말고 모자이크 또는 재촬영.
  • 03 🧹 토큰 청소 · 세 곳 · 필수 →
    Claude Code mcp remove · GHL "16강-실습용" Delete · 메모장 PIT 줄. 세 곳 모두 비워야 오늘 마무리.
