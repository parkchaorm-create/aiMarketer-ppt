Phase 4 첫 회차 · 세 가지

  • 01 🗺️ Phase 3→4 지도 · 한 장 이해 →
    내 컴퓨터 안에서만 움직이던 Claude Code(Anthropic의 터미널 기반 AI 코딩 에이전트)가 외부 CRM 시스템과 악수하는 단계로 넘어갑니다.
  • 02 🎯 오늘 목표 · 연결 한 가지 →
    GHL(GoHighLevel · 에이전시용 올인원 CRM+마케팅 자동화 플랫폼) 가입과 MCP(Model Context Protocol · AI와 외부 도구를 연결하는 표준 규약) 연결 확인 한 번만.
  • 03 🧳 준비물 · 네 가지 체크 →
    카드 · 이메일 · 터미널 환경 · Claude Pro 플랜. 카드 없으신 분은 옆자리 화면 공유로 참여 가능합니다.

AI가 바깥과 · 손 내민 적 있나요?

  • 01 💻 Phase 3 · 내 컴퓨터 안에서만 →
    터미널을 열고 CLAUDE.md(Claude Code가 자동으로 읽는 프로젝트 안내서 파일)를 쓰고 슬래시 명령어를 만들었어요. 유용했지만 한 가지 한계가 있었습니다.
  • 02 🏢 고객 데이터 · 내 컴퓨터 밖 →
    실제 고객 데이터는 내 컴퓨터 밖 CRM(Customer Relationship Management · 고객 관계 관리) 시스템에 있어요. 손이 닿지 않았던 지점입니다.
  • 03 🤝 오늘 · 경계를 넘는 첫 악수 →
    오늘은 그 경계를 넘는 날. 본격 자동화는 17강부터. 오늘은 "악수의 첫 순간" 한 가지에만 집중합니다.

Phase 3 완료 · Phase 4 출발

  • 01 🏠 Phase 3(10~15강) · 로컬 완료 →

    오늘 하는 한 번의 연결이 17~25강 아홉 회차 전부의 기반이 됩니다. 욕심 없이 "연결 한 가지"에만.

04 / 07

오늘 책상 위 · 네 가지

  • 01 💳 카드 · 평가판 가입용 →
    GHL 14일 평가판 등록에 해외결제 가능한 체크·신용카드. 14일 안에 해지하시면 청구 없음. 카드 없으시면 옆자리 화면 공유로 참여.
  • 02 📧 이메일 · 인증용 한 개 →
    지메일 또는 사용 중인 업무 이메일. GHL이 인증 메일을 보내므로 받을 수 있는 주소여야 합니다.
  • 03 ⌨ 터미널 환경 · 10강 IDE →
    10강에서 설치한 안티그래비티 IDE 또는 동등한 터미널이 작동해야 합니다. Claude Code가 설치된 작업 폴더도 함께.
  • 04 🔐 Claude Pro · 유지 확인 →
    5강에서 만드신 Claude Pro 플랜이 유지되는지 확인. 운영진이 어제 #공지에 카드 등록 안내를 이미 올렸습니다.
