MCP. 지난주까지 수업에서 한 번도 나온 적 없는 세 글자입니다. 오늘부터 9주 동안(17~25강) 거의 매 회차 나올 단어이기도 해요. 한 번 정확하게 짚고 가겠습니다.

MCP(Model Context Protocol · AI 모델이 외부 도구·데이터와 연결되는 표준 규약)는 2024년 11월 Anthropic이 공개한 오픈 프로토콜입니다[^3]. 오픈 프로토콜이라는 말은 "특정 회사의 독점 기술이 아니라, 누구나 이 규약에 맞추기만 하면 연결할 수 있는 공용 규격"이라는 뜻이에요.

  • 03 🔑 PIT · 스페어 열쇠 한 개 →

OVERVIEW
📝 강사 노트
💬 그대로 읽으세요
현재 Claude Code·Cursor·Windsurf·Claude Desktop 등 여러 AI 코딩 도구가 이 규약을 지원합니다. GHL도 2026년 초부터 공식 MCP 서버를 운영하고 있어요[^4].

컴퓨터에 USB 허브를 꽂으면 마우스·키보드·외장하드·프린터를 한 포트로 연결할 수 있죠. MCP는 AI의 USB 허브입니다. Claude Code라는 AI 본체에 MCP라는 허브를 달면, GHL·Google Calendar·Slack·Notion 같은 외부 도구들을 하나의 규격으로 꽂을 수 있어요.

지난주까지 Phase 3에서 Claude Code는 내 컴퓨터 파일만 읽을 수 있었습니다. 오늘부터는 MCP 허브를 통해 외부 클라우드 서비스도 직접 읽고 쓸 수 있어요.

MCP · 지난주 한 번도 안 나온 세 글자?

  • 01 🆕 오늘 처음 · 내일부턴 매주 →
    MCP. 지난주까지 수업에서 한 번도 나온 적 없는 세 글자. 17~25강 9주 동안 거의 매 회차 나올 단어입니다.
  • 02 🧱 지난주까진 · 내 파일만 →
    Phase 3 Claude Code는 내 컴퓨터 파일만 읽을 수 있었어요. 외부 서비스 연결은 늘 "통역 코드를 따로 짜야" 했습니다.
  • 03 🤝 오늘부턴 · 외부 클라우드까지 →
    MCP가 "표준 규약"이 된 덕에 · Claude Code · Cursor · Windsurf 모두 같은 방식으로 · 외부 도구와 악수할 수 있게 됐어요.
HOOK
📝 강사 노트
💬 그대로 읽으세요
Claude Code는 AI의 언어로 말하고, GHL은 REST API라는 개발자 언어로 말합니다. 예전에는 이 둘을 연결하려면 직접 중간 통역 코드를 짜야 했어요. MCP는 그 통역사 역할의 표준입니다. 한 번 세워 두면, Claude Code가 자연어로 "내 연락처 목록 보여줘"라고 말했을 때, MCP가 자동으로 GHL이 이해하는 API 요청으로 번역해 줍니다.

MCP 허브를 꽂으려면 인증이 필요해요. "이 AI가 정말 내 GHL에 접근해도 되는 AI인가?"를 확인하는 절차입니다. 이때 쓰는 것이 PIT(Private Integration Token · 개인 전용 연결 열쇠)입니다[^5].

PIT를 비유하면 "내 GHL 집의 스페어 열쇠 한 개" 입니다. 다음 네 가지 성질이 중요해요.

USB 허브 · 통역사 · 두 얼굴

  • 01 🔌 은유 1 · AI의 USB 허브 →
    컴퓨터 USB 허브가 마우스·키보드를 한 포트로 묶듯 MCP는 AI 본체에 GHL·Slack·Notion 같은 외부 도구를 한 규격으로 꽂아 줍니다.
  • 02 💬 은유 2 · 언어 사이의 통역사 →
    Claude Code는 자연어, GHL은 API라는 개발자 언어. MCP는 그 사이 통역사. "연락처 수 세어줘" → 자동으로 API 요청으로 번역.
  • 03 🌐 오픈 · 특정 회사 독점 아님 →
    MCP는 Anthropic이 공개한 오픈 프로토콜. Cursor · Windsurf · Claude Desktop 등 누구나 같은 규약으로 연결 가능합니다.
CONCEPT · 01
📝 강사 노트
💬 그대로 읽으세요
오늘 Claude Code 허브에 꽂을 MCP 서버는 딱 하나 — `ghl` 서버입니다. GHL 공식 서버의 endpoint(접속 주소)는 다음과 같아요[^4].

https://services.leadconnectorhq.com/mcp/

이 주소에 PIT(인증 열쇠)와 Location ID(어느 서브계정인지)를 함께 보내면, GHL이 Claude Code에게 약 36가지 도구를 열어 줍니다(연락처·대화·파이프라인·캘린더·이메일·블로그 등). 로드맵상 250개 이상으로 확장 예정이에요.

04 / 07

PIT · 내 GHL 집의 스페어 열쇠

  • 01 📏 긴 문자열 · 60자 무작위 →
    60자 이상 무작위 영문·숫자 조합. 한 번 생성하면 다시 볼 수 없으니 즉시 메모장에 복사해 두어야 합니다.
  • 02 🎯 scope 제한 · 5개만 →
    허용한 scope(권한 범위)만 접근해요. 수업용은 연락처·대화·기회·장소 읽기 위주 5개만 체크. 실수로 고객 데이터 건드릴 위험 최소화.
  • 03 🎫 Bearer 토큰 · 은행 비밀번호 →
    OAuth(OAuth · 외부 앱 인증 표준)의 Bearer 토큰 형식. 슬랙·카톡·이메일·화면 공유에 절대 노출 금지.
  • 04 🔄 수동 순환 · 수업 후 삭제 →
    90일마다 교체 권장이 원칙. 수업용 토큰은 "그날 쓰고 그날 삭제"가 원칙. Part 06에서 다 같이 삭제합니다.
CONCEPT · 02
📝 강사 노트
💬 그대로 읽으세요
Part 05에서 실제로 입력하실 명령어는 단 3줄입니다. 지금 외우실 필요는 없고, 한 번 눈에 익혀만 두세요.

claude mcp add ghl --transport http \ --header "Authorization: Bearer <내_PIT>" \ --header "locationId: <내_LOCATION_ID>" \ https://services.leadconnectorhq.com/mcp/

Part 03 · 네 줄 정리

  • 01 📘 MCP · AI와 외부 도구 표준 →
    Anthropic 2024년 공개 오픈 프로토콜. AI 모델이 외부 도구·데이터와 연결되는 표준 규약. 여러 AI 도구가 공통 지원.
  • 02 🔌 두 얼굴 · USB 허브·통역사 →
    하드웨어 은유=AI의 USB 허브. 언어 은유=자연어↔API 통역사. 둘 다 "표준 규격으로 연결" 감각.
  • 03 🔑 PIT · 스페어 열쇠 · 오늘 삭제 →
    긴 문자열·scope 5개·Bearer 토큰·오늘 쓰고 삭제. Claude Code 명령어는 단 3줄(add·list·remove)입니다.
