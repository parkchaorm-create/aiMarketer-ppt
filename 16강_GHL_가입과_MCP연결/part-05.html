💬 그대로 읽으세요
10강에서 설치한 안티그래비티 IDE 또는 본인이 쓰시는 터미널을 여세요. Claude Code가 설치된 작업 폴더로 이동한 뒤, 아래 명령어를 입력합니다.

claude mcp add ghl \ --transport http \ --scope user \ --header "Authorization: Bearer <여기에_본인_PIT_붙여넣기>" \ --header "locationId: <여기에_본인_LOCATION_ID_붙여넣기>" \ https://services.leadconnectorhq.com/mcp/

02 / 07

가입 화면이 · 영어라 · 겁나시나요?

  • 01 🇰🇷 UI · 한국어 없이 영어만 →
    GHL은 영어 전용 UI. 한국어 설정 없어요. 크롬의 "한국어로 번역" 기능과 튜토리얼 한국어 주석 스크린샷이 보조 역할.
  • 02 💳 카드 · 14일 무료 보장 →
    14일 동안 결제 없음. 14일 안에 해지하시면 청구 0원. 카드 등록은 평가판 확인 용도일 뿐입니다.
  • 03 👥 카드 없으시면 · 옆자리 공유 →
    카드 없으시면 팀원·옆자리 수강생 화면 공유로 함께. 수업 진행에 전혀 지장 없습니다. 오늘은 "흐름 한 번 보기".

계정 · Location · 순서대로

  • 01 💼 업체명 · 팀 없으면 본인 이름 →
    아직 에이전시 이름 없으시면 "(본인 이름) 마케팅 스튜디오". 언제든 수정 가능. 전화번호는 +82 국가 코드.
  • 02 📧 인증 이메일 · 받은편지함 확인 →
    가입 후 이메일이 오면 링크 클릭 → 로그인. 스팸함에 갔는지도 확인. 인증 없으면 로그인 불가.
  • 03 🏢 Location · 서브계정 작업 공간 →
    온보딩 마법사: Location 이름 "16강_실습_(본인)" · 업종 Marketing Agency · 국가 South Korea · 시간대 Asia/Seoul.
  • 04 🔗 Location ID · URL에서 복사 →
    생성된 Location에 들어가서 URL의 app.gohighlevel.com/v2/location/긴문자열/ 부분 긴 문자열이 Location ID. 메모장에 임시 붙임.

PIT 발급 · 7단계 + 보안 경고

  • 01 🧭 Settings → Private Integrations →
    만든 Location 안에서 좌측 메뉴 맨 아래 Settings → 왼쪽 탭 리스트 Private Integrations → 우측 Create New Integration.
  • 02 🏷️ 이름 필수 · "16강-실습용" →
    토큰 이름은 반드시 "16강-실습용-(본인이름)". Part 06에서 일괄 삭제할 때 식별용입니다.
  • 03 🎯 scope · 5개만 체크 →
    contacts.readonly · contacts.write · conversations.readonly · opportunities.readonly · locations.readonly. 실수로 고객 데이터 건드릴 위험 거의 0.
  • 04 🔒 Create · 즉시 복사 · 메모장만 →
    Create 누르면 토큰 문자열 한 번 뜨고 화면 닫으면 다시 못 봅니다. 슬랙·이메일 업로드 금지. 메모장 임시 파일에만.
