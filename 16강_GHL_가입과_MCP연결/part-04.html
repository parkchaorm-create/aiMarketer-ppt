  • 01 🚪 문 1 · 계정 생성 + 인증 →
    gohighlevel.com에서 Start 14-Day Free Trial → Starter $97 → 이메일·이름·비밀번호·업체명·카드 입력 → 이메일 인증.

Location은 서브계정이라고 생각하시면 됩니다. 에이전시가 클라이언트마다 하나씩 만드는 작업 공간이에요. 오늘은 본인 실습용 하나만.

만든 Location 안에서:

1. 좌측 메뉴 맨 아래 Settings 2. 왼쪽 탭 리스트에서 Private Integrations 3. 우측 상단 Create New Integration 버튼 4. 이름: "16강-실습용-(본인이름)" (필수 · 나중에 일괄 삭제용) 5. Scope 선택: 아래 5개만 체크 (수업용 최소)

`contacts.readonly`

endpoint · 도구 36+ · 3줄 명령

  • 01 📡 GHL MCP endpoint · 공식 →
    services.leadconnectorhq.com/mcp/ · GHL 공식 MCP 서버 주소. Claude Code가 이 한 주소로 접속합니다.
  • 02 🧰 열리는 도구 · 약 36가지 →
    연락처·대화·파이프라인·캘린더·이메일·블로그·소셜미디어 등 약 36가지 도구. 로드맵상 250+ 확장 예정.
  • 03 ⌨ add · 서버 추가 한 줄 →
    claude mcp add ghl --transport http --header "Authorization: Bearer 내PIT" --header "locationId: 내LOC" endpoint주소 — 한 번만.
  • 04 📋 list · remove · 나머지 두 줄 →
    claude mcp list(목록 확인) · claude mcp remove ghl(수업 후 제거). 전부 합쳐 단 3줄입니다.
