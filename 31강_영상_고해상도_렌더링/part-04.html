로그인 · 다운로드까지

  • 01 1️⃣ 로그인 · 26강 그 계정 →
    higgsfield.ai 접속 · 26강에서 개설한 계정 그대로 로그인. 우측 상단 크레딧 잔액을 시연 시작 전에 한 번 확인합니다.
  • 02 2️⃣ Upscale 페이지 · 직접 URL →
    상단 메뉴 Video Upscale 또는 직접 higgsfield.ai/upscale 입력. 페이지 진입하면 4 모델 카드가 보입니다.
  • 03 3️⃣ 모델 선택 · Theia 카드 클릭 →
    Theia 카드 클릭 → 업로드 화면 전환. Rhea·Gaia·Focus Fix는 카드만 가리키면서 "오늘은 Theia"라고 짧게 짚습니다.
  • 04 4️⃣ 파일 업로드 · 드래그앤드롭 →
    30강 원본 영상을 드래그앤드롭 또는 Upload 버튼. 100MB 이내 권장. 큰 파일은 미리 분할 안내.
  • 05 5️⃣ 옵션 · 기본값 유지 →
    샤프닝·디테일 파라미터 3종 수동 조정 가능하지만 기본값 유지 권장. 첫 실습은 변경 없이 가는 것이 안전합니다.
  • 06 6️⃣7️⃣ Generate · 대기·다운로드 →
    크레딧 차감 확인하며 Generate 클릭. 몇 분 후 Before/After 비교 뷰로 결과 확인 후 다운로드. 시연 한 사이클 종료.

미리 알면 · 당황 X

  • 01 ⛔ ① 업로드 중간 멈춤 · 파일 크기 →
    파일 100MB 초과 시 흔합니다. 앞·뒤 잘라내기로 축약 + 네트워크 상태 확인. 필요시 무료 온라인 툴(clipchamp 등)로 분할.
  • 02 🚫 ② Generate 비활성 · 크레딧 부족 →
    우측 상단 크레딧 잔액 확인. 부족하면 Theia보다 더 저렴한 옵션은 없으므로 강사 화면 공유로 대체 참여하시면 됩니다.
  • 03 ⏰ ③ 5분 초과 · 서버 상황 →
    서버 부하에 따라 변동 가능. 10분 넘어가면 강사에게 알려 주세요. 그동안 다른 팀원 체크리스트 작업을 병렬로 돌립니다.
  • 04 🤔 ④ 결과·원본 거의 같음 · 자연 →
    원본이 이미 1080p+면 Theia 효과가 제한적입니다. 이 경우 원본 그대로 최종본으로 확정 가능 · 실패가 아닙니다.
  • 05 😅 ⑤ 차이 작음 · 실망 금지 →
    Theia는 샤프닝·디테일 강화 중심. 드라마틱한 변화가 아닌 은은한 품질 개선이 기본 기대치입니다. "완벽한 복원"은 아닙니다.
