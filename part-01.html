18분 · 클로드 코드 정체 + 요금제

  • 01🎻 이론 선생 vs 실전 악단 · 차이 완벽 이해→
    ChatGPT는 음악 이론 선생님 — "이렇게 하세요" 설명만. Claude Code는 직접 연주하는 악단 — 파일 생성·스캔·저장까지 실행합니다.
  • 02💳 Free / Pro / Max · 어떤 플랜이 내 돈값?→
    Free=견학만(Claude Code 사용 불가), Pro=$20/월=왕초보 추천, Max=$25~100/월=헤비유저용. 결론: Pro로 시작하세요.
  • 03🚕 토큰 · 택시 미터기 원리 + 절약 꿀팁 3가지→
    대화 길수록 미터기 상승. 꿀팁: ① /clear로 리셋 ② 작업별 새 세션 ③ 가벼운 작업은 Haiku 모델(경차 택시).

02 / 10

설명만 vs 직접 실행

  • 01💬 ChatGPT · "SEO 분석해줘" → 방법만 알려줌→
    "SEO를 분석하려면 이런 걸 확인하셔야 합니다"라고 방법을 알려주죠. 음악 선생님이 "이 곡은 이렇게 연주하면 좋습니다"라고 설명하는 것과 같습니다.
  • 02🎻 Claude Code · 같은 질문 → 스캔 → 리포트 파일 저장→
    실제로 블로그를 스캔하고, 문제점 리스트를 만들고, 수정 방법까지 파일로 저장합니다. 악단이 직접 연주해서 녹음까지 해주는 거죠.
  • 03📂 내 PC에서 실행 · 파일 접근 + 수정 + 생성 가능→
    Claude Code는 여러분 컴퓨터 안에서 실행됩니다. 그래서 파일에 직접 접근해서 수정할 수 있는 거죠.
  • 04🏠 악단이 내 거실에 상주 · 필요할 때 즉시 연주→
    오케스트라로 치면, 악단이 우리 집 거실에 상주하는 겁니다. 필요할 때마다 내 악보(파일)를 가져다가 직접 연주해주니까요.
  • 05⚡ 핵심 · "설명" vs "실행" = 진짜 자동화의 차이→
    챗봇은 "이렇게 하세요"라고 설명만, Claude Code는 직접 파일 만들어서 저장까지. 이게 진짜 자동화입니다.

사업자용 활용법 10가지

  • 01🎬 콘텐츠 3종 · 소셜미디어 + 뉴스레터 + 영상 스크립트→
    URL 하나 넣으면 인스타·유튜브·블로그 동시 생산. 뉴스레터는 브랜드 보이스 맞춤, 스크립트도 뚝딱.
  • 02🔍 분석 4종 · SEO 감사 + GEO(AI 검색) + GA4 + 경쟁사→
    SEO 문제점 우선순위 정리, GEO로 AI 검색 노출 최적화, GA4 인사이트 한국어 요약, 경쟁사 비교 분석까지.
  • 03🤖 오케스트라 2종 · 멀티 에이전트 팀 + 자동 트리거→
    리서처·카피라이터·SEO AI를 팀으로 편성. 매주 월요일 자동 콘텐츠 계획도 가능.
  • 04🛠️ 실용 2종 · 이메일 A/B 테스트 + 랜딩페이지 최적화→
    시퀀스 이메일 초안부터 A/B 문구 작성, 랜딩페이지 전환율 높이는 카피 수정·코드 수정까지.
  • 05🏁 오늘 악기를 하나씩 만들고 · 마지막에 합주!→
    10가지 중 핵심을 직접 실습합니다. 악기 하나씩 만들고, 마지막에 전부 합주하는 거죠!

    전 세계에서 한국 사용자 성장률이 가장 높습니다. 데이터가 증명하죠.

    나중에는 뒷줄에서 서서 볼 수도 있습니다. 지금이 최고의 타이밍이에요.

요금제 · 공연장 대관료

  • 01💰 월 $20(약 3만 원) · 무대 빌려야 공연 가능→
    3만 원이 오케스트라 공연장 대관료입니다. 공연장 없이는 악단이 연주할 무대가 없잖아요?
  • 02🚫 Free · Claude Code 사용 불가 · 견학만→
    무료라서 매력적이지만 치명적 단점 — Claude Code를 사용할 수 없습니다. 공연장 견학만 가능.
  • 03⭐ Pro $20/월 · 전체 기능 · 왕초보 추천→
    Claude Code 전체 기능 사용 가능. 하루에 상당한 양을 작업할 수 있습니다. 오늘 시작할 플랜.
  • 04🏟️ Max $25~100/월 · 헤비유저용 · 지금은 불필요→
    매일 몇 시간씩 돌리는 헤비유저용. 여러 단계가 있지만 왕초보 단계에서는 필요 없습니다.
  • 05✅ 결론 · Pro로 시작하세요→
    왕초보 단계에서는 Pro로 충분합니다. 한국 카드 전부 결제 가능.

토큰 · 택시 미터기

  • 01🚕 대화 길이 = 이동 거리 · 길수록 미터기 상승→
    택시 타면 이동 거리에 따라 미터기가 올라가듯, AI도 대화가 길수록 토큰이 올라갑니다.
  • 02📏 내 질문 + AI 답변 · 양쪽 다 토큰 소모→
    제가 말을 많이 하고, AI가 답을 많이 할수록 미터기(토큰)가 올라갑니다.
  • 03⏸️ Pro · 하루 주행 한도 있음→
    Pro 플랜은 하루에 탈 수 있는 택시 거리가 정해져 있습니다.
  • 04🐢 한도 초과 · 속도 저하 또는 일시 중지→
    한도를 초과하면 그날은 속도가 느려지거나 일시 중지됩니다.
  • 05📅 팁 · 몰아쓰기 금지 · 하루 1~2시간 꾸준히→
    작업을 몰아서 하지 말고 하루 1~2시간씩 꾸준히 쓰는 게 좋습니다.

Pro 구독 vs API 직접 호출

  • 01📞 API · AI한테 직접 전화 · 주문 전화번호→
    API는 쉽게 말해 음식점 주문 전화번호. 배달 앱 대신 직접 전화 걸어서 주문하는 방식입니다.
  • 02⚙️ 설정 복잡 + 비용 예측 불가 + 폭탄 위험→
    설정이 복잡하고, 비용이 얼마나 나올지 예측도 어렵고, 실수로 비용 폭탄 맞는 경우도 있습니다.
  • 03⚠️ 왕초보 · API 직접 사용 비추→
    왕초보한테 API 직접 사용은 비추입니다. 리스크 대비 이점이 없어요.
  • 04✅ Pro 구독 · 월정액 택시 · 편하고 안전→
    Pro 구독으로 Claude Code 쓰는 게 훨씬 편하고 안전합니다. 월 정액제 택시라고 생각하세요.
  • 05🎯 결론 · Pro로 시작이 정답→
    나중에 사용량이 늘면 그때 API나 Max를 고려해도 늦지 않습니다.

토큰 절약 꿀팁 3선

  • 01🔄 /clear · 대화 리셋 = 미터기 초기화→
    /clear 명령어로 대화를 주기적으로 초기화. 미터기를 리셋하는 거라고 생각하세요.
  • 02🚕 작업별 새 세션 · 새 택시 타기→
    작업별로 새 세션을 시작하세요. 새 택시 타는 겁니다. 이전 대화 맥락이 토큰을 잡아먹지 않아요.
  • 03⚡ 가벼운 작업 · Haiku 모델 = 경차 택시→
    가벼운 작업은 Haiku 모델을 쓰세요. 경차 택시라서 기본료가 쌉니다. 뒤에서 더 자세히 다룹니다.
  • 04💡 원리 · 말 길수록 토큰 상승 · 짧고 명확하게→
    프롬프트를 짧고 명확하게 쓰면 토큰을 절약할 수 있습니다. 장문보다 핵심만.

Part 01 · 요약 세 줄

  • 01🎻 Claude Code · 설명이 아닌 실행 · 직접 연주하는 악단→
    ChatGPT는 설명만 해주는 선생님. Claude Code는 직접 파일을 만들고 실행까지 해주는 악단입니다.
  • 02💰 Pro $20/월로 시작 · 왕초보에겐 이것으로 충분→
    Free는 사용 불가, Max는 아직 불필요. Pro 플랜 월 3만 원이 정답입니다.
  • 03🔄 토큰 = 택시 미터기 · /clear 리셋으로 아껴 쓰기→
    토큰은 택시 미터기. /clear로 리셋, 작업별 세션 분리, Haiku 활용으로 절약하세요.
