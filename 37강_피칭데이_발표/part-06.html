Phase 1~6 · 핵심 산출물

  • 01 📘 P1·P2 · 매뉴얼 + 포트폴리오 →
    Phase 1(1~4강 · 최미혜): 팀 브랜드 매뉴얼 1권. Phase 2(5~9강 · 김국희): 팀 포트폴리오 PDF + 웹 시안. 에이전시 소개·영업 자료의 뼈대.
  • 02 ⚙️ P3·P4 · 자동화 + GHL →
    Phase 3(10~15강 · 최미혜): 팀 콘텐츠 자동 에이전트 + 리포트. Phase 4(16~25강 · 김국희): 팀 GHL(Go High Level · 마케팅 통합 플랫폼) 파이프라인. 에이전시 기본 엔진.
  • 03 🎬 P5·P6 · 영상 + 피칭 →
    Phase 5(26~31강 · 최미혜): 팀 영상 30초+ 1편. Phase 6(32~37강 · 김국희): 포트폴리오 웹·스케줄링·제안서·피칭 영상(오늘). 라이브 발표 경험 포함.
  • 04 🔄 재활용 · 오늘 이후 어디에 →
    매뉴얼은 에이전시 소개서. 포트폴리오는 영업 자료. 자동화는 고객 대행 핵심. 영상은 SNS 대표 소재. 매뉴얼이 그대로 끝이 아닙니다.
